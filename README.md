townsquare-flask
================


Setup
=====

Make sure you are using Python 3.5! This app does not make any attempt
to be backwards compatible with older versions.

```
git clone git@github.com:sc3/townsquare-flask.git

cd townsquare-flask

mkvirtualenv town

pip install -r requirements.txt

```

Run the app:

```
./application.py

```

Run the tests:

```
nosetests tests

```
